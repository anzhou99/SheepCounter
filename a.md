## 曲线路径的动画实现

首先就是小羊跳入、跳出界面的动画，仔细观察地话可以发现，小羊移动的路径其实是条曲线。

水平或者垂直移动的动画不难实现，我们可以通过控制绝对定位下的 left/right 或者 top/bottom 来实现。

```css
@keyframe moveX {
    0% {
        top: 0;
    }
    100% {
        top: 100px;
    }
}
.moveX:hover {
    position: absolute;
    top: 0;
    animation: moveX 1s;
}
```
